# 原生HTML+CSS+JS音乐播放器

​		本项目使用原生HTML+CSS+JS实现了一个web端音乐播放器，未使用任何框架，具体实现功能如下：

1. 进度条功能
2. 时间功能
3. 播放/暂停功能
4. 上一首/下一首
5. 切换播放模式功能
6. 音量调节功能
7. 播放MV功能
8. 倍速功能
9. 音乐列表功能

## 页面设计

本项目页面设计参考QQ音乐播放界面，整体色调以暗绿色为主。
