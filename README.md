# whwdzg.github.io

个人向导览与存档站点（2.0）。

## 在线访问
- 主页：https://whwdzg.github.io
- 关于页面：/about.html
- 旧版存档：/Legacy-1.0/

## 功能概览
- 单页导航：共享的 header/sidebar/aside 壳层与轻量路由
- 多语言与主题切换：预置多语种 JSON 与深浅色模式
- 渐进式增强：懒加载图片、滚动返回顶部、设置面板
- PWA：manifest、service worker 与简单离线缓存

## 目录速览
- /css：全站样式与变量
- /js：交互脚本、PWA、辅助工具
- /includes：站点公共壳层片段
- /Legacy-1.0：旧版本备份与回顾

## 本地预览
仓库是纯静态站点，无需构建步骤。确保已安装 Python 3 后，在项目根目录运行：

```
python serve.py
```

脚本会启动简易本地服务器，便于预览。
